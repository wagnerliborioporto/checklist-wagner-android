# Checklist Wagner — Android

Aplicativo Android que abre o Checklist Wagner hospedado no GitHub Pages.

## Gerar o APK pelo GitHub

1. Envie todos os arquivos deste projeto para um repositório.
2. Abra a aba **Actions**.
3. Entre em **Gerar APK**.
4. Clique em **Run workflow**.
5. Ao terminar, baixe o artefato **checklist-wagner-apk**.

O arquivo dentro do ZIP será `app-debug.apk`.
